# QAgent

AI-driven end-to-end test runner. Goals are written in natural language; an LLM picks actions, Playwright drives the browser, and verifiers are pure code.

**Status:** experimental, pre-CLI. The runner is currently invoked via `node src/demo.js "<goal>"`. The CLI is being designed in [`docs/cli-approach.md`](docs/cli-approach.md).
